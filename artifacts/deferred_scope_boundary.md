# Deferred Scope Boundary

The shipped flagship version stops at a deterministic, facility-side, outpatient-first control room with thin follow-through, what-if, downstream-evidence, and memo support.

It does not ship:

- Predictive triage as the operating center
- Denials workflow, appeals operations, or payer adjudication behavior
- Full correction / rebill operations platform depth
- New departments or service lines beyond the frozen V1 set
- Broad workflow-engine or task-management expansion
- Production integration claims

If a future enhancement weakens those boundaries, it is outside this release candidate.
